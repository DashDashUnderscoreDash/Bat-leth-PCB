# **Bat'leth**

Bat'leth is an STM32 10u treadstone unibody split PCB with leafsprings tadpole mounting

It has a forced layout and is forced solder


### In this repo you will find:
```
- Layout options
- Renders of PCB
- STEP model of PCB
- Edgecuts

Firmware and real photos WIP as this has not been produced as of 5/2/26. I am ordering a batch to verify all works shortly and will update the repo
```
